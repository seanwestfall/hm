# hm
An implementation of the Hindley-Milner type system ([wikipedia](https://en.wikipedia.org/wiki/Hindley%E2%80%93Milner_type_system)) in a simple Scheme/Lisp like language.

# build
```bash
# ghc -o dist/hm --make src/hs.hs
# ./dist/hm ...
```
